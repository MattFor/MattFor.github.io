# Upcoming changes

Here's what is currently planned or actively being worked on.

Public release of the code for one, and then:

## Dashboard
A web-based management dashboard for Relaxy! Server owners will be able to configure the bot, view stats and manage features without needing to type a single command.

## Self-hosted website
Moving the website off GitHub Pages and onto a self-hosted server for full control over routing and caching.

## Bug extermination
A thorough pass over the codebase to track down and close every known issue before the next version bump.

## Structural optimisation
Consolidating the database schema and cleaning up how the bot's internals are organised. It will be leaner, faster, easier to extend.

## Music revival
The music module is getting a full rework. Better stream stability, improved queue management and proper error recovery.

and much.. much more!
